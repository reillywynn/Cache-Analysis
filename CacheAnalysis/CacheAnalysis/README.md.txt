Click run
It will ask for input for block size, associativity, and cache size as well as the name of the file and the replacement policy.
The parameters are in powers of 2 (2^(user input)). Ex: for 64 byte block size enter 6
Once you enter the parameters and the filename and click run it will take a couple seconds to run then display the results. 

*Newest version of C++ visual studios required, C++ 20
